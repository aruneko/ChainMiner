# ChainMiner
## これなに
いわゆる鉱石の一括採掘機能を追加する。Spigot用プラグインです。

## (動作|作成)環境
- Spigot 1.16.5
- Java 1.8
- Kotlin 1.4.30

## How To Build
以下のコマンドを入力すると、`build/libs/`以下にjarファイルができあがります。

```bash
$ ./gradlew build
```

## 使い方
スニーク状態で一括採掘機能が有効になります。各種ツルハシで以下のものが一括採掘可能です。

- 石炭
- ダイヤ
- エメラルド
- 金
- 鉄
- ラピス
- ネザークォーツ
- レッドストーン
- 古代の残骸
